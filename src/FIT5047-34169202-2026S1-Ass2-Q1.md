**Definition Recap: Logical Status**

* **Valid (Tautology):** True in **all** possible models.
* **Not Valid:** False in **at least one** model.
* **Satisfiable:** True in **at least one** models.
* **Unsatisfiable (Contradiction):** False in **all** possible models.
* **Satisfiable but Not Valid:** True in some models, but false in others.

</div>

#### 1. Formula $\phi_1$
$$ \phi_1 = ((A \lor (B \rightarrow C)) \land (\neg C \lor D)) \rightarrow ((A \rightarrow D) \lor (C \leftrightarrow E)) $$

*   **Satisfiability Check:** 
    Let $A = False$ and $D = True$. The Right-Hand Side (RHS) becomes $(False \rightarrow True) \vee \dots = True$. Since the RHS of the main implication is True, the entire formula $\phi_1$ is True. Thus, it is **satisfiable**.
*   **Validity Check:** 
    To disprove validity, we attempt to find a model where $\phi_1$ is False (i.e., LHS is True and RHS is False).
    *   For RHS to be False: Both $(A \rightarrow D)$ and $(C \leftrightarrow E)$ must be False. 
        *   $(A \rightarrow D) = False \implies A = True, D = False$.
        *   $(C \leftrightarrow E) = False \implies C \neq E$.
    *   For LHS to be True (given $A=T, D=F$):
        *   $(\neg C \lor D) \implies (\neg C \lor F) \implies \neg C$ must be True $\implies C = False$.
        *   Since $C = False$ and $C \neq E$, we must have $E = True$.
        *   The first part of LHS is $(A \lor (B \rightarrow C)) = (True \lor \dots) = True$.
    *   Using the model $\{A=T, B=T, C=F, D=F, E=T\}$, LHS is True and RHS is False, resulting in $True \rightarrow False \equiv False$.

🎯 **Conclusion for $\phi_1$:** The formula is **Satisfiable but Not Valid**. We have identified at least one interpretation where it holds true and one counter-interpretation where it holds false.

*   **Satisfiability Check:**
    We can satisfy $\phi_2$ by making the LHS False. Let $A = False$ and $E = True$. The term $(\neg E \lor A) = (False \lor False) = False$. Since LHS is False, the main implication evaluates to True. Thus, it is **satisfiable**.
*   **Validity Check:**

    *   Let $A = False$ and $E = False$. LHS becomes $((False) \lor (C \rightarrow D)) \land (True) \equiv (C \rightarrow D)$.

    *   Model: $\{A=F, B=T, C=F, D=F, E=F, F=F\}$. LHS is True, RHS is False.

**Definition Recap: CNF and Horn Clauses**

* **CNF Conversion:** Any logical sentence can be converted into **Conjunctive Normal Form (CNF)**, a conjunction of disjunctions (AND of ORs) by applying four theoretical steps:
    1.  **Eliminate biconditionals $\leftrightarrow$:** $(A \leftrightarrow B) \equiv (A \to B) \land (B \to A)$
    2.  **Eliminate implications $\to$:** $(A \to B) \equiv (\neg A \lor B)$
    3.  **Move $\neg$ inwards:** Apply **De Morgan's laws** and **Double Negation**.
    4.  **Distribute $\lor$ over $\land$:** $A \lor (B \land C) \equiv (A \lor B) \land (A \lor C)$

* **Horn Clause:** A clause (disjunction of literals) containing **at most one positive literal**. These are critical for efficient inference in logical agents.

</div>

#### 1. Transformation of $\phi_1$
$\phi_{1} = ((A \vee (B \rightarrow C)) \wedge (\neg C \vee D)) \rightarrow ((A \rightarrow D) \vee (C \leftrightarrow E))$

2.  **Move negations inwards (De Morgan's):** 
    * $\neg LHS = (\neg A \wedge B \wedge \neg C) \vee (C \wedge \neg D)$

3.  **Simplify and apply absorption laws:**
    Combine $\neg LHS \vee RHS$:
    * $= (\neg A \wedge B \wedge \neg C) \vee (C \wedge \neg D) \vee \neg A \vee D \vee ((\neg C \vee E) \wedge (\neg E \vee C))$

      * The presence of $\vee D$ absorbs $\neg D$ leaving $C \vee D$.

5.   **Final CNF:** $\mathbf{(\neg A \vee C \vee D \vee \neg E)}$
6.   **Horn Clauses:** **None**. The single clause contains two positive literals ($C, D$), violating the Horn rule.

**Definition Recap: Implications and Inference**

* **Vacuous Truth (Implications):** An implication ($P \rightarrow Q$) is evaluated as True unless the premise ($P$) is True and the conclusion ($Q$) is False. If the premise is False, the implication is **vacuously True**.
* **Logical Inference (Entailment):** We say $\alpha \models \beta$ if $\beta$ is True in every **interpretation** where $\alpha$ is True. If satisfying models exist with conflicting truth values for a variable, no definitive entailment exists.

</div>

Given $\phi_{2} = (((\neg A \wedge B) \vee (C \rightarrow D)) \wedge (\neg E \vee A)) \rightarrow ((\neg D \vee F) \leftrightarrow (A \wedge B))$. 

Assume $F = False$.

#### (i) Effect on Satisfiability
Substituting:
* $F = False$ into $\phi_2$, the RHS $(\neg D \vee F) \leftrightarrow (A \wedge B)$ 

Smplifies to:
* $ (\neg D \lor False) \leftrightarrow (A \land B) \equiv \neg D \leftrightarrow (A \land B) $.

The modified formula $\phi_{2}'$ is: 
* $ \phi_2' = \text{LHS} \rightarrow (\neg D \leftrightarrow (A \land B)) $.
