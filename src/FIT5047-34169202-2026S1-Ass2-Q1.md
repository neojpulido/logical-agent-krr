### Part (a): Validity and Satisfiability

**Definition Recap:**
*   **Valid:** True in *all* possible models (tautology).
*   **Satisfiable but not valid:** True in *some* models, but false in others.
*   **Unsatisfiable:** False in *all* possible models (contradiction).

#### 1. Formula $\phi_1$
$$ \phi_1 = ((A \lor (B \rightarrow C)) \land (\neg C \lor D)) \rightarrow ((A \rightarrow D) \lor (C \leftrightarrow E)) $$

*   **Satisfiability Check:** 
    Let $A = \text{False}$ and $D = \text{True}$. The Right-Hand Side (RHS) becomes $(F \rightarrow T) \lor \dots = \text{True} \lor \dots = \text{True}$. Since the RHS of the main implication is True, the entire formula $\phi_1$ is True. Thus, it is **satisfiable**.
*   **Validity Check:** 
    To disprove validity, we attempt to find a model where $\phi_1$ is False (i.e., LHS is True and RHS is False).
    *   For RHS to be False: Both $(A \rightarrow D)$ and $(C \leftrightarrow E)$ must be False. 
        *   $(A \rightarrow D) = \text{False} \implies A = \text{True}, D = \text{False}$.
        *   $(C \leftrightarrow E) = \text{False} \implies C \neq E$.
    *   For LHS to be True (given $A=T, D=F$):
        *   $(\neg C \lor D) \implies (\neg C \lor F) \implies \neg C$ must be True $\implies C = \text{False}$.
        *   Since $C = \text{False}$ and $C \neq E$, we must have $E = \text{True}$.
        *   The first part of LHS is $(A \lor (B \rightarrow C)) = (\text{True} \lor \dots) = \text{True}$.
    *   Using the model $\{A=T, B=T, C=F, D=F, E=T\}$, LHS is True and RHS is False, resulting in $\text{True} \rightarrow \text{False} \equiv \text{False}$.

> **Conclusion for $\phi_1$:** It is **satisfiable but not valid**.

#### 2. Formula $\phi_2$
$$ \phi_2 = (((\neg A \land B) \lor (C \rightarrow D)) \land (\neg E \lor A)) \rightarrow ((\neg D \lor F) \leftrightarrow (A \land B)) $$

*   **Satisfiability Check:**
    We can satisfy $\phi_2$ by making the LHS False. Let $A = \text{False}$ and $E = \text{True}$. The term $(\neg E \lor A) = (\text{False} \lor \text{False}) = \text{False}$. Since LHS is False, the main implication evaluates to True. Thus, it is **satisfiable**.
*   **Validity Check:**
    Let's find a counterexample where LHS is True and RHS is False.
    *   Let $A = \text{False}$ and $E = \text{False}$. LHS becomes $((\text{False}) \lor (C \rightarrow D)) \land (\text{True}) \equiv (C \rightarrow D)$.
    *   For RHS to be False, $(\neg D \lor F) \leftrightarrow (\text{False} \land B) \implies (\neg D \lor F) \leftrightarrow \text{False}$. This requires $(\neg D \lor F)$ to be True.
    *   Let $D = \text{False}$ and $F = \text{False}$. Then $(\neg D \lor F) = (\text{True} \lor \text{False}) = \text{True}$. RHS evaluates to $\text{True} \leftrightarrow \text{False} \equiv \text{False}$.
    *   For LHS to remain True, $(C \rightarrow D)$ must be True. Given $D=\text{False}$, $C$ must be $\text{False}$. Let $B = \text{True}$.
    *   Model: $\{A=F, B=T, C=F, D=F, E=F, F=F\}$. LHS is True, RHS is False.

> **Conclusion for $\phi_2$:** It is **satisfiable but not valid**.

#### 3. Formula $\phi_3$
$$ \phi_3 = ((A \leftrightarrow B) \lor (C \rightarrow D)) \land ((\neg A \lor \neg D) \rightarrow (F \land \neg G)) $$

*   **Satisfiability Check:**
    Let $A = T, B = T, D = T$.
    *   Left conjunct: $(T \leftrightarrow T) \lor \dots = \text{True}$.
    *   Right conjunct antecedent: $(\neg T \lor \neg T) = \text{False}$. The implication $\text{False} \rightarrow \dots$ is $\text{True}$.
    *   Both conjuncts are True, making $\phi_3$ True. Thus, it is **satisfiable**.
*   **Validity Check:**
    We can make $\phi_3$ False by simply making the left conjunct False. Let $A = T, B = F$ (so $A \leftrightarrow B$ is False) and $C = T, D = F$ (so $C \rightarrow D$ is False). The left conjunct is False, making the entire formula False.

> **Conclusion for $\phi_3$:** It is **satisfiable but not valid**.

### Part (b): CNF Transformation and Horn Clauses

A clause is a **Horn clause** if it contains *at most one positive literal*.

#### 1. Transformation of $\phi_1$
$$ \phi_1 = ((A \lor (B \rightarrow C)) \land (\neg C \lor D)) \rightarrow ((A \rightarrow D) \lor (C \leftrightarrow E)) $$
$$ \phi_1 = \neg ( (A \lor \neg B \lor C) \land (\neg C \lor D) ) \lor ( (\neg A \lor D) \lor ( (C \lor \neg E) \land (\neg C \lor E) ) ) $$

Applying De Morgan's on the LHS and rewriting:
*   $\neg \text{LHS} = (\neg A \land B \land \neg C) \lor (C \land \neg D)$
*   $\text{RHS} = (\neg A \lor D \lor C \lor \neg E) \land (\neg A \lor D \lor \neg C \lor E)$

Distributing $\neg \text{LHS}$ over the RHS conjunctions ($\phi_1 = (\neg \text{LHS} \lor \text{RHS}_1) \land (\neg \text{LHS} \lor \text{RHS}_2)$):
1.  **First Conjunct:** $(\neg A \land B \land \neg C) \lor (C \land \neg D) \lor (\neg A \lor D \lor C \lor \neg E)$
    *   $(C \land \neg D) \lor D \equiv C \lor D$.
    *   The clause collapses to: $\neg A \lor C \lor D \lor \neg E$.
2.  **Second Conjunct:** $(\neg A \land B \land \neg C) \lor (C \land \neg D) \lor (\neg A \lor D \lor \neg C \lor E)$
    *   $(C \land \neg D) \lor \neg C \equiv \neg C \lor \neg D$.
    *   Contains $D \lor \neg D \equiv \text{True}$.

**Final CNF for $\phi_1$:**
$$ (\neg A \lor C \lor D \lor \neg E) $$
*   **Horn Clause Check:** Contains two positive literals ($C, D$). **Not a Horn clause.**

#### 2. Transformation of $\phi_2$
Convert $\phi_2 = \text{LHS} \rightarrow \text{RHS}$ into $\neg \text{LHS} \lor \text{RHS}$.
*   $\neg \text{LHS} = ( (A \lor \neg B) \land C \land \neg D ) \lor (E \land \neg A) \equiv (A \land C \land \neg D) \lor (\neg B \land C \land \neg D) \lor (E \land \neg A)$
*   $\text{RHS} = (D \lor A) \land (D \lor B) \land (\neg F \lor A) \land (\neg F \lor B) \land (\neg A \lor \neg B \lor \neg D \lor F)$

Distributing $\neg \text{LHS}$ over the 5 clauses of the RHS and applying absorption laws yields 11 distinct clauses:
1.  $ \neg A \lor \neg B \lor \neg D \lor F $
2.  $ \neg B \lor A \lor D \lor E $
3.  $ A \lor C \lor D \lor E $
4.  $ B \lor C \lor D \lor E $
5.  $ \neg A \lor B \lor C \lor D $
6.  $ A \lor C \lor E \lor \neg F $
7.  $ A \lor \neg D \lor E \lor \neg F $
8.  $ B \lor C \lor E \lor \neg F $
9.  $ \neg A \lor B \lor C \lor \neg F $
10. $ B \lor \neg D \lor E \lor \neg F $
11. $ \neg A \lor B \lor \neg D \lor \neg F $

**Horn Clauses in $\phi_2$:**
*   Clause 1: $ \neg A \lor \neg B \lor \neg D \lor F $ (One positive literal: $F$) 
*   Clause 11: $ \neg A \lor B \lor \neg D \lor \neg F $ (One positive literal: $B$)

#### 3. Transformation of $\phi_3$
$$ \phi_3 = ((A \leftrightarrow B) \lor (C \rightarrow D)) \land ((\neg A \lor \neg D) \rightarrow (F \land \neg G)) $$
Break into conjuncts and distribute:
*   **Conjunct 1:** $( (A \lor \neg B) \land (\neg A \lor B) ) \lor \neg C \lor D \equiv (A \lor \neg B \lor \neg C \lor D) \land (\neg A \lor B \lor \neg C \lor D)$
*   **Conjunct 2:** $\neg(\neg A \lor \neg D) \lor (F \land \neg G) \equiv (A \land D) \lor (F \land \neg G)$
    *   Distribute OR over AND: $(A \lor F) \land (A \lor \neg G) \land (D \lor F) \land (D \lor \neg G)$

**Final CNF for $\phi_3$:**
1.  $ A \lor \neg B \lor \neg C \lor D $
2.  $ \neg A \lor B \lor \neg C \lor D $
3.  $ A \lor F $
4.  $ A \lor \neg G $
5.  $ D \lor F $
6.  $ D \lor \neg G $

**Horn Clauses in $\phi_3$:**
*   Clause 4: $ A \lor \neg G $ (One positive literal: $A$)
*   Clause 6: $ D \lor \neg G $ (One positive literal: $D$)

### Part (c): Evaluating $\phi_2$ when $F$ is always False

Assume $F = \text{False}$.

#### (i) Effect on Satisfiability (4 marks)
Substituting $F = \text{False}$ into $\phi_2$, the RHS $(\neg D \lor F) \leftrightarrow (A \land B)$ simplifies to:
$$ (\neg D \lor \text{False}) \leftrightarrow (A \land B) \equiv \neg D \leftrightarrow (A \land B) $$
The modified formula $\phi_2'$ is:
$$ \phi_2' = \text{LHS} \rightarrow (\neg D \leftrightarrow (A \land B)) $$
**Result:** The formula **remains satisfiable**. We can still easily satisfy it by making the LHS False (e.g., setting $A = \text{False}$ and $E = \text{True}$ guarantees LHS is False, thereby making the implication True, regardless of $D$ or $B$).

#### (ii) Can you infer anything about A or B? Justify.
In propositional logic, to "infer" $A$ (i.e., $\phi_2' \models A$), $A$ must be True in *all* satisfying models of $\phi_2'$. 

Let's test different satisfying models for $\phi_2'$:
*   **Model 1 ($A$ is False, $B$ is True):** Let $A = F, E = T$. The LHS evaluates to False. The implication holds True. 
*   **Model 2 ($A$ is True, $B$ is True):** Let $A = T, B = T, C = F, D = F, E = T$. 
    *   LHS: $( (\text{False}) \lor (F \rightarrow F) ) \land (\text{False} \lor T) \equiv \text{True} \land \text{True} \equiv \text{True}$.
    *   RHS: $\neg \text{False} \leftrightarrow (\text{True} \land \text{True}) \equiv \text{True} \leftrightarrow \text{True} \equiv \text{True}$.
    *   $\phi_2'$ holds True.
*   **Model 3 ($A$ is True, $B$ is False):** Let $A = T, B = F, C = F, D = T, E = T$. 
    *   LHS: $( (\text{False}) \lor (F \rightarrow T) ) \land (\text{False} \lor T) \equiv \text{True}$.
    *   RHS: $\neg \text{True} \leftrightarrow (\text{True} \land \text{False}) \equiv \text{False} \leftrightarrow \text{False} \equiv \text{True}$.
    *   $\phi_2'$ holds True.

**Conclusion:** 
No, we **cannot unconditionally infer anything** about the truth values of $A$ or $B$. As demonstrated above, there exist valid, satisfying models where $A$ and $B$ can independently take either True or False values. The knowledge base ($\phi_2'$) is not strict enough to entail a specific truth assignment for $A$ or $B$.
